## Installation

To install the `IlmTest CLI`, ensure you have Node.js version 23.0.0 or later.

```bash
npm install -g ilmtest-cli
```

Or

```bash
npx ilmtest-cli
```

Or

````bash
bunx ilmtest-cli
```

## Requirements
````
